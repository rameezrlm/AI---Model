## API
**Definition:** A set of rules and tools allowing software applications to communicate and share data seamlessly.
## SDK
**Definition:** A collection of tools, libraries, and documentation for building software applications on specific platforms.
## Web APIs
**Definition:** Interfaces enabling interaction with web-based services over HTTP protocols.
## Library APIs
**Definition:** Pre-written code modules providing specific functionalities to simplify software development.
## Prompting and Completion
**Definition:** Interaction model where inputs (prompts) generate relevant outputs (completions) from AI systems.
